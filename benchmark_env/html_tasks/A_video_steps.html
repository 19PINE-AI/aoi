Installation Steps (Reference)

1

Download

Download the Arcturus Platform installer from the official releases page. Choose the binary for your operating system.

2

Install

Run the installer package. The setup wizard guides you through license agreement, installation directory, and component selection.

3

Configure

Open the configuration panel and set your environment variables, database connection, and initial workspace settings.
