Color System

Nexus Systems uses a carefully curated color palette that communicates trust, technical precision, and forward motion. Our primary brand color anchors all of our visual communications. 

/* Nexus Systems Brand Colors */
--ns-primary:     #1a73e8;   /* Primary brand color */
--ns-primary-dark:#1557b0;   /* Hover / active states */
--ns-text:        #202124;   /* Body text */
--ns-surface:     #f8f9fa;   /* Background surface */
--ns-border:      #dadce0;   /* Dividers and borders */

The hexagonal logo mark (the "NS" hex) must always appear in the primary brand color on white backgrounds, and in white on dark backgrounds. Never use alternative colors for the logo without explicit brand team approval.
